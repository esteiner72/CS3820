# CS3820
Semester AI project by Ethan Steiner and Kaleb Stamey

Requirements:
- Python 3.10+
- Node.js v19.1.0+

User instructions to run locally:
1. Clone the CS3820 repository to your computer
2. Open the repository (main branch) in your IDE or navigate with the cd command in a terminal
3. Install node dependecies using the command: npm ci
4. Start the server:
   - (Windows) npx nodemon
   - (Mac/Linux) nodemon
5. Frontend will be available at http://localhost:3000
